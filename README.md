# TopStories

TopStories app uses local JSON that is read via the Bundle class and parses the contents into Swift objects using Codable and JSONDecoder()

## Apple APIs used

1. JSONDecoder
1. Codable
1. URL
1. URLSession
1. Result type 
1. Grand Central Dispatch (GCD)
